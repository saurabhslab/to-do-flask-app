# To-Do App (Flask)

A simple To-Do task manager built using Flask.

## Features
- Add new tasks
- View all tasks
- Delete tasks

## Project Structure
todo_app/
├── app.py
├── templates/
│   └── index.html
└── static/
    └── style.css

## How to Run the Project

1. Install Flask
pip install flask

2. Run the application
python app.py

3. Open browser and visit
http://127.0.0.1:5000/

## How It Works
- Tasks are stored in a Python list
- Form is used to add tasks
- Each task has a delete button

## Technologies Used
- Python
- Flask
- HTML
- CSS
